```bash
skillrank create [options]
```

Interactive wizard for building a stack manifest. Walks you through naming, selecting skills, and picking target agents.

## Usage

```bash
skillrank create
```

You'll be prompted through:

<Steps>
  <Step title="Stack name">
    Default: current directory name.
  </Step>
  <Step title="Author">
    Default: your `git config user.name`.
  </Step>
  <Step title="Description">
    A one-line description of what this stack is for.
  </Step>
  <Step title="Target agents">
    Multi-select from your detected agents.
  </Step>
  <Step title="Pick skills">
    Multi-select from your installed skills.
  </Step>
  <Step title="Add remote skills (optional)">
    Add skills from any GitHub repo by entering `owner/repo` and skill name.
  </Step>
</Steps>

The result is a `skillrank.yaml` written to the current directory.

## Options

<ParamField path="-n, --name <name>">
  Pre-fill the stack name.
</ParamField>

<ParamField path="-o, --output <path>" default="./skillrank.yaml">
  Output path for the generated file.
</ParamField>
